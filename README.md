# 🎓 Student Grade Calculator (Python Project)

A simple Python-based console application that calculates a student's total marks, percentage, and grade based on input marks for multiple subjects.

---

## 🚀 Features

- Takes student name as input
- Accepts marks for 5 subjects
- Calculates:
  - Total marks
  - Percentage
  - Grade based on performance
- Handles invalid input using error handling

---

## 📊 Grade System

| Percentage | Grade |
|------------|------|
| 90 - 100   | A+   |
| 80 - 89    | A    |
| 70 - 79    | B    |
| 60 - 69    | C    |
| 50 - 59    | D    |
| Below 50   | Fail |

---

## 🧑‍💻 Technologies Used

- Python 3
- Basic input/output handling
- Conditional statements
- Exception handling (try-except)
